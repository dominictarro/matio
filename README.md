# matio
Library for fast input/output of sparse datasets.

### Requirements
------------
<a href='https://www.python.org/downloads/release/python-380/'>Python 3.8/3.7</a>
<a href='https://pypi.org/project/numpy/'>NumPy</a>

### Supported Methods
------------
- Dictionary of Keys: 'dok'
- Coordinate List:    'coo'
